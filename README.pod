=head1 PostgreSQL Mac Preferences

=head2 Overview

This project provides a Mac OS X System Preferences pane for the PostgreSQL Database Server.

=head2 Install Instructions

Download and double-click this file to install in System preferences. Alternatively, download to a folder
then drag it onto the System Preferences icon.

=head2 Uninstall Instructions

Right-click on the PostgreSQL icon in System Preferences, and click "Remove".

Also, delete the following files if they exist:

=over

=item *

/Library/LaunchAgents/com.hkwebentrepreneurs.postgresql.plist*

=item *

~/Library/LaunchAgents/com.hkwebentrepreneurs.postgresql.plist*

=back
